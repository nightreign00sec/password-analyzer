# password-analyzer

A tool to evaluate password hygiene. It accepts a password, calculates entropy, estimates the time to crack it, and reports other specific weaknesses.

## Features

- Password entropy calculation
- Estimated crack time
- Detection of common weaknesses

## Usage

1. Provide a password to the analyzer.
2. Review entropy and strength results.
3. Fix detected weaknesses to improve security.

## Purpose

This project helps users understand how secure their passwords are and encourages better password practices.
